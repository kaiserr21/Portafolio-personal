# Proyectos personales en Python

-CALCULADORA
Este programa se encarga de resolver operaciones basicas tales como suma, resta, multiplicacion y division
tiene una operatividad sencilla 

-CONVERTIDOR DE UNIDADES
Este pequeño programa pude convertir las siguientes unidades:
- Celsius a Fahrenheit
- Kilómetros a millas
- Kilogramos a libras
- Altura en metros a formato USA (5'11")

-GENERADOR DE CONTRASEÑAS
Este es un programa hecho en Python que genera contraseñas seguras y evalúa su nivel de seguridad.
Consiste en generar una contraseña aleatoria, por consiguiente hace una validacion de que tan segura es
y por ultimo la clasifica en Debil, Media, Fuerte
En este programa se puede elegir la longitud de la contraseña
