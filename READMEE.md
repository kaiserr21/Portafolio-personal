# Mi portafolio web
Este proyecto contiene mi portafolio web donde presento mis proyectos personales de programación y participación en hackathon
Mi portafolio está diseñado para mostrar:

- Proyectos personales en Python.
- Participación en hackathon
- Habilidades técnicas en programación y desarrollo web.
- Enlaces a mis repositorios en GitHub.

El portafolio está desarrollado en **HTML y CSS**

## 🛠 Tecnologías utilizadas

- **HTML5**: estructura del sitio web.
- **CSS3**: estilos y diseño visual.
- **GitHub**: alojamiento de código.
- **GitHub Pages**: publicación del portafolio como sitio web público
